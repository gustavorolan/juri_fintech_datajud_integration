## Instalação

```bash
$ npm install
```

## Rodar o app

```bash
# Comando para subir postgres no docker
$ docker compose up 
```

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Rodar Testes

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e
```
